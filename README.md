# Behaviourial Trends Among Investors
## Statement 
This project explores the <a href = "https://www.kaggle.com/datasets/ayeshasiddiqa123/finance-trends-2020-2025">Finance_Trends.csv</a> dataset using Python, Pandas, Matplotlib, and Seaborn to analyze investor behavior and preferences across different financial products by examining the relationships. The goal is to uncover actionable insights that can guide personalized financial recommendations and improve segmentation strategies for investor outreach."

---

## Project Overview

The notebook `prototype.ipynb` performs a full **Exploratory Data Analysis (EDA)** that includes:
- Loading and inspecting the dataset
- Understanding column-level information
- Generating visualizations
- Computing percentages and distributions
- Writing insights for each section
